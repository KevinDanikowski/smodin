***Using pm2 when running from create-react-app***
starting: `pm2 start node_modules/react-scripts/scripts/start.js --name react-app`
stopping: `pm2 stop react-app` or `pm2 stop all`
restart: `pm2 restart react-app` or `pm2 restart all`

***Get rid of port number with lib2cap-bin***
download: 'sudo setcap cap_net_bind_service=+ep `readlink -f \`which node\``'
edit file: go to node_modules/react-scripts/scripts/start.js and change port from 3000 to 80

******* New Install ******
get SSH key from computer: cat ~/.ssh/id_rsa.pub
create droplet, add SSH
connect: ssh root@IPADDRESS
update packages: sudo apt-get update
install npm: sudo apt-get install npm
install node: curl -sL https://deb.nodesource.com/setup_9.x | sudo -E bash -
              sudo apt-get install -y nodejs
enable firewall: sudo ufw enable (this can cause problems with logging in, you can skip this)
install nginx: sudo apt-get install nginx
allow nginx firewall: sudo ufw allow 'Nginx Full'
change nginx file: cd etc/nginx/site-available
    make sure to add proxy location to change ports if you run the port on 3000 usually or something to redirect to 80
    SAMPLE:
    server {
            listen 80;
            root /var/www/smodin/public;
            index index.html index.htm;
            server_name smodin.me;
            location / {
            proxy_pass http://0.0.0.0:3000;
            include /etc/nginx/proxy_params;
            }
        }
restart nginx: sudo service nginx restart
copy over config files (this could include .babelrc, etc.)
install pm2 to run server: npm i pm2 -g
start server: pm2 start file_or_file_path --name what_to_call_it
    if you use create-react-app, use this path: pm2 start node_modules/react-scripts/scripts/start.js --name app
